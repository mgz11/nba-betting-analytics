# Sports Betting Analytics

Full-stack NBA betting analytics platform analyzing how teams perform relative to sportsbook expectations.

## Tech Stack

- Next.js
- TypeScript
- Python ETL
- PostgreSQL
- Recharts

## Current Progress

✅ Local odds JSON ingestion

✅ Python ETL pipeline

✅ PostgreSQL storage

✅ Next.js API integration

✅ Basic dashboard rendering

## Run Project

### Frontend

```bash
cd frontend
npm install
npm run dev
```
